# Images Folder

This folder contains all the photos and videos for the Sofia collection site.

## How to Add Images

1. Upload your image files (JPG, PNG, etc.) to this folder
2. Update the `index.html` file to reference them:

```html
<img src="images/your-image-name.jpg" alt="Description">
```
